## Data Flow — Video Generation

```
User clicks "Generate"
        ?
        ?
POST /generate-video
        ?
        ???? Create job doc in Cosmos (status=queued, pipeline_step=queued)
        ?
        ???? Dispatch Celery task: run_video_pipeline(job_id, config, user_id)
        ?
        ?
   CELERY WORKER
        ?
        ???? [1] STORY:  OpenAI GPT-4o ? StoryResult (title, scenes, text)
        ?         ?? checkpoint.json saved
        ?         ?? Cosmos: pipeline_step = "generating_story"
        ?
        ???? [2] IMAGES: OpenAI DALL-E 3 × N scenes ? scene_000.png, ...
        ?         ?? checkpoint.json saved (per-image resume on crash)
        ?         ?? Cosmos: pipeline_step = "generating_images"
        ?
        ???? [3] NARRATION: OpenAI TTS-1 ? narration.mp3
        ?         ?? checkpoint.json saved
        ?         ?? Cosmos: pipeline_step = "generating_narration"
        ?
        ???? [4] SUBTITLES: Scene text ? story.srt
        ?         ?? checkpoint.json saved
        ?         ?? Cosmos: pipeline_step = "generating_subtitles"
        ?
        ???? [5] ASSEMBLY: MoviePy + FFmpeg ? story_YYYYMMDD_HHMMSS.mp4
        ?         ?? checkpoint.json saved
        ?         ?? Cosmos: pipeline_step = "assembling_video"
        ?
        ???? [6] BLOB UPLOAD: Azure Blob Storage (4 MB chunks, 3 retries)
        ?         ?? checkpoint.json saved
        ?         ?? Cosmos: pipeline_step = "uploading_blob"
        ?
        ???? [7] CLEANUP: Delete local images, audio, subtitles, video
        ?         ?? checkpoint.json deleted
        ?         ?? Cosmos: status=completed, pipeline_step=done
        ?
        ?
   BLOB = Single Source of Truth
        ?
        ???? Frontend plays from blob URL
        ???? YouTube upload downloads from blob URL
```

## Authentication Flow

```
POST /auth/login { email, password }
        ?
        ???? Query users container by email
        ???? bcrypt.checkpw(password, hash)
        ???? Generate JWT (user_id, email, role, exp=24h)
        ???? Return { access_token, token_type: "bearer" }

All protected routes:
        ?
        ???? HTTPBearer extracts token from Authorization header
        ???? jwt.decode(token, secret, HS256)
        ???? Returns { user_id, email, role } as current user
```

## Checkpoint / Resume System

```
assets/job_<uuid>/
??? checkpoint.json          ? Pipeline state (step, story, paths)
??? images/
?   ??? scene_000.png
?   ??? scene_001.png
?   ??? ...
??? audio/
?   ??? narration.mp3
??? subtitles/
    ??? story.srt

On crash:
  1. Worker restarts
  2. load_checkpoint(work_dir) reads checkpoint.json
  3. step_already_done() checks each step
  4. Skips completed steps ? resumes from failure point
  5. Missing images detected ? only regenerates those

On success:
  1. Entire work_dir deleted (shutil.rmtree)
  2. Local video deleted
  3. Only blob URL persists
```
